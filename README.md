# Project
Анализ системы метрик приложения после внедрения нового вида контента.
Задача: Провести анализ юнит-экономики продукта в динамике за первые 4 недели.
Разобраться в причинах стагнации выручки. Определить, прибыльные источники
привлечения и какие не выходят на уровень окупаемости.
Результаты: Данные загружены, предобработанны и подготовлены для кагорного анализа.
Рассчитаны и проанализированы метрики вовлечённости в продукт. Рассчитаны и
проанализированы финансовые метрики CAC, LTV и ROI за первые 4 недели с момента
привлечения. Исследованы метрики в разрезе источника привлечения. Рассчитан период
окупаемости маркетинговых вложений.
Артефакты: Сформулировать выводы в виде аналитической записки с рекомендациями,
методикой расчета и визуализацией.
Стек: SQL (PostgreSQL), Python (Pandas, Matplotlib, Seaborn), Jupyter Notebook, DBeaver.
